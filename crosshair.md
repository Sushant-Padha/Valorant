# CROSSHAIR
Format (copy and edit):
```
Outlines										 : On|Off
Center Dot (Opacity, Thickness)					 : - -
Inner Lines	(Opacity, Length, Thickness, Offset) : - - - -
Outer Lines	(Opacity, Length, Thickness, Offset) : - - - -
```
